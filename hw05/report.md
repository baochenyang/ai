# 实验五：极简 CNN 手写数字识别与 LeNet-5 实现

## 一、文章学习摘要

参考公众号文章《计算机视觉》第10篇：极简卷积神经网络 CNN 识别手写数字。

### 文章核心内容

1. **CNN 基本结构**
   - 卷积层（Conv2D）：提取图像特征
   - 池化层（MaxPooling2D）：降维，减少参数量
   - 全连接层（Dense）：分类输出

2. **MNIST 数据集**
   - 60,000 张训练图像，10,000 张测试图像
   - 每张图像 28×28 像素，灰度图
   - 标签为 0-9 的数字

3. **训练流程**
   - 数据归一化（像素值缩放到 [0,1]）
   - 编译模型（选择优化器 Adam 和损失函数）
   - 训练模型（反向传播更新权重）
   - 评估模型（计算测试集准确率）

4. **模型结构**
   - Conv2D(32, 3×3) + MaxPooling2D(2×2)
   - Conv2D(64, 3×3) + MaxPooling2D(2×2)
   - Conv2D(64, 3×3)
   - Flatten() + Dense(64) + Dense(10)

文章来源：参考课程提供的公众号文章


## 二、LeNet-5 结构说明

### 网络结构表

| 层 | 类型 | 核大小/神经元 | 输出尺寸 | 参数量 |
|---|------|--------------|----------|--------|
| 输入 | - | - | 32×32×1 | 0 |
| C1 | 卷积 | 5×5, 6个 | 28×28×6 | 156 |
| S2 | 平均池化 | 2×2 | 14×14×6 | 0 |
| C3 | 卷积 | 5×5, 16个 | 10×10×16 | 2,416 |
| S4 | 平均池化 | 2×2 | 5×5×16 | 0 |
| C5 | 卷积 | 5×5, 120个 | 1×1×120 | 48,120 |
| F6 | 全连接 | 84个 | 84 | 10,164 |
| 输出 | 全连接 | 10个 | 10 | 850 |

**总参数量**：约 61,706

### 结构特点
- 使用 tanh 激活函数（原始 LeNet-5 风格）
- 使用平均池化（AveragePooling2D）
- 逐层降低空间尺寸，增加通道数
