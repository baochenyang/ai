## 五、测试集结果

### 5.1 评估指标

| 指标 | 值 |
|------|-----|
| 准确率 (Accuracy) | **96.79%** |
| 精确率 (Precision) | **97.94%** |
| 召回率 (Recall) | **97.44%** |
| F1分数 | **97.69%** |

### 5.2 混淆矩阵

| | 预测正常 | 预测肺炎 |
|---|----------|----------|
| 实际正常 | 226 | 8 |
| 实际肺炎 | 10 | 380 |

**计算过程**：
- 测试集共 624 张：正常 234 张，肺炎 390 张
- 正确分类：正常 226 张，肺炎 380 张
- 错误分类：假阳性 8 张，假阴性 10 张

### 5.3 指标计算详解

- **准确率** = (226 + 380) / 624 = 606 / 624 = **0.9679** (96.79%)
- **精确率** = 380 / (380 + 8) = 380 / 388 = **0.9794** (97.94%)
- **召回率** = 380 / (380 + 10) = 380 / 390 = **0.9744** (97.44%)
- **F1分数** = 2 × (0.9794 × 0.9744) / (0.9794 + 0.9744) = **0.9769** (97.69%)

## 六、结果分析

### 1. 准确率 vs 召回率

从结果看，**召回率（97.44%）略低于精确率（97.94%）**，两者接近，说明模型在准确性和覆盖度之间取得了良好平衡。

**从医学诊断角度，召回率更重要**：

| 错误类型 | 后果 | 严重程度 |
|----------|------|----------|
| **假阴性**（肺炎→正常） | 患者延误治疗，病情加重，可能危及生命 | ⭐⭐⭐⭐⭐ |
| **假阳性**（正常→肺炎） | 患者接受不必要的检查和心理负担，医疗资源浪费 | ⭐⭐⭐ |

因此，在医疗影像诊断中，宁可误诊不可漏诊，**高召回率应优先考虑**。本模型召回率 97.44%，意味着每 100 个肺炎患者中约 97 个能被正确识别，临床价值较高。

### 2. 数据增强的作用

本实验使用了以下数据增强技术：
- 随机水平翻转（p=0.5）
- 随机旋转（±10度）

观察训练曲线：
- 训练准确率从 90% 提升到 98%
- 验证准确率稳定在 96-97%，与训练准确率差距小

**结论**：数据增强有效缓解了过拟合，提升了模型泛化能力。

### 3. 假阴性 vs 假阳性的影响

本实验混淆矩阵显示：
- **假阴性（FN）= 10**：10 张肺炎被误判为正常
- **假阳性（FP）= 8**：8 张正常被误判为肺炎

从临床角度：
- 10 个假阴性可能导致 10 名患者延误治疗，是更严重的问题
- 8 个假阳性会导致不必要的焦虑和检查，但不会危及生命

**改进方向**：进一步提高召回率，可尝试降低分类阈值或对肺炎类别加权。

### 4. 模型改进建议

1. **迁移学习**：使用 ResNet50、VGG16 等预训练模型，提取更丰富的特征
2. **注意力机制**：添加 SE 模块或自注意力，聚焦病灶区域
3. **类别加权**：对肺炎类别增加权重，降低假阴性率
4. **集成学习**：多个模型投票，提高预测稳定性

## 九、结论

本实验成功实现了胸部X光肺炎影像的二分类任务：

1. **模型表现**：测试集准确率 96.79%，召回率 97.44%，F1分数 97.69%
2. **医学意义**：高召回率符合临床需求，漏诊率仅 2.56%
3. **技术价值**：轻量级CNN（仅26万参数）即可取得 96%+ 准确率，适合资源受限场景
4. **改进方向**：可尝试迁移学习、注意力机制进一步提升性能

肺炎影像分类是一项**高召回率优先**的任务，本模型在召回率和准确率之间取得了良好平衡，具有实际临床参考价值。
